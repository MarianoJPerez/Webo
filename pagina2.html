Diagramas de Estados

1. Producto cargado: El fabricante/proveedor interesado en que Blanco reseñe su producto realiza la primera solicitud y el producto es cargado al sistema.

2. Entrega de producto: Una vez cargado el producto, Blanco se comunica con el fabricante/proveedor para coordinar la entrega de producto a reseñar.

  • 2.1. Producto solicitado: El producto se solicita al fabricante/proveedor.
  • 2.2. Enviado: El fabricante/proveedor envió el producto.
  • 2.3. Producto recibido: Blanco recibió el producto.

3. Pendiente de asignación: El producto se encuentra pendiente de asignación al influencer indicado.

4. Asignado: El producto ya ha sido asignado a un influencer.

5. Envio de producto a influencer: Una vez asignado el influencer que hará la reseña, Blanco le debe hacer llegar el producto.

6. Unboxing pendiente: El influencer ya recibió el producto y tiene que realizar el unboxing.

7. Unboxing publicado: El influencer publicó el unboxing por video/streaming.

8. En prueba por influencer: El influencer está probando el producto por unos días para luego publicar la reseña final.

9. Producto reseñado: El influencer finalizó la prueba del producto y publicó la reseña.
